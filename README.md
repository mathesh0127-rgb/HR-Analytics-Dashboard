
HR Analytics Dashboard | Power BI

📌 Project Overview

Developed an interactive HR Analytics Dashboard using IBM HR Analytics dataset to analyze employee attrition, retention, overtime impact, and workforce trends.

📊 Key Features

* KPI cards for Attrition Rate, Retention Rate, Active Employees
* Department-wise attrition analysis
* Overtime impact analysis
* Trend analysis using line and bar charts
* Interactive slicers and filters

🛠 Tools Used

* Power BI
* DAX
* Power Query
* Data Visualization

📈 Insights

* Employees with overtime showed higher attrition
* Younger employees had higher turnover
* Certain departments showed increased attrition trends
